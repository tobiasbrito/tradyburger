Politica de privacidad

Tradi Burgerrr utiliza los datos enviados por sus clientes unicamente para tomar pedidos, coordinar entregas, responder consultas y mejorar la atencion comercial.

Datos que podemos solicitar

Podemos solicitar nombre, telefono, direccion de entrega, detalle del pedido, forma de pago y notas necesarias para preparar o despachar el pedido.

Uso de WhatsApp

Si el cliente escribe por WhatsApp, el sistema puede responder automaticamente para mostrar el menu, tomar el pedido y enviar el resumen al local. Los mensajes se usan solo para la gestion del pedido.

Base de datos

Los pedidos pueden almacenarse en una base de datos privada para que el comercio pueda ver el historial, confirmar estados y organizar la preparacion.

Contacto

Para solicitar correcciones o eliminacion de datos relacionados con un pedido, el cliente puede comunicarse con el comercio por sus canales oficiales.
